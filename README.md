# 🚗 Cars24 Used Cars Data Analysis (Web Scraping + EDA)

## 📌 Project Overview

This project focuses on collecting and analyzing used car data from the **Cars24 online marketplace** through web scraping and performing **Exploratory Data Analysis (EDA)** to understand patterns in used car pricing.

The goal of the project is to identify the factors that influence the resale value of cars such as **car brand, fuel type, kilometers driven, transmission type, and manufacturing year**.

This project demonstrates practical skills in **web scraping, data cleaning, data analysis, and data visualization**, which are essential for Data Analyst roles.

---

## 🎯 Objectives

* Scrape used car listings from Cars24 website
* Build a structured dataset of car attributes
* Clean and preprocess the collected data
* Perform exploratory data analysis (EDA)
* Identify trends affecting used car prices
* Visualize insights using Python libraries

## 📊 Dataset Features

The scraped dataset contains the following features:

* Car Name / Model
* Brand
* Manufacturing Year
* Selling Price
* Kilometers Driven
* Fuel Type (Petrol / Diesel / CNG)
* Transmission Type (Manual / Automatic)
* Owner Type
* Location

---

## 🔎 Exploratory Data Analysis

The following analyses were performed:

* Distribution of car prices
* Price comparison across fuel types
* Relationship between car age and price
* Impact of kilometers driven on resale value
* Most popular car brands in the used car market
* Transmission type vs price analysis

---

## 📈 Key Insights

* Older cars generally have lower resale prices.
* Cars with fewer kilometers driven tend to have higher market value.
* Manual transmission cars are more common in the used car market.
* Certain brands maintain higher resale value than others.
